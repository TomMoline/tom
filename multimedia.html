Interactive Multimedia

This is an example of an interactive element that was created in Tumult Hype. It demonstrates rollover actions and mouse click actions. Although not well known, Tumult Hype is a program that creates animations and interactive content that works on desktops, smartphones and iPads with little Javascript coding required. This example shows the features of Tolomatic's new TRS electric screw-drive rodless actuator and is optimized for desktop and tablets.
